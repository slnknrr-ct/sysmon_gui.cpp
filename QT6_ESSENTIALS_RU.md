# Основы Qt 6.10.1 для проекта SysMon3

Этот документ описывает основные возможности и концепции Qt 6.10.1, используемые в GUI-приложении SysMon3.

## Основные возможности Qt 6

### 1. Модуль Core (QtCore)

#### Объектная модель
- **QObject** - Базовый класс для всех объектов Qt
- **Макрос Q_OBJECT** для системы мета-объектов
- **Сигналы и слоты** - Событийно-ориентированная коммуникация
- **Отношения родитель-потомок** - Автоматическое управление памятью

#### Система мета-объектов
- **QMetaObject** - Информация о типах времени выполнения
- **qobject_cast<T>()** - Безопасное приведение типов QObject
- **Динамические свойства** - Управление свойствами времени выполнения

#### Работа с потоками
- **QThread** - Управление потоками
- **QTimer** - Операции на основе таймеров
- **QMutex** - Синхронизация потоков
- **QAtomic** - Атомарные операции

#### Классы-контейнеры
- **QString** - Работа с Unicode строками
- **QByteArray** - Операции с байтовыми массивами
- **QList<T>** - Динамические массивы
- **QMap<K,V>** - Пары ключ-значение
- **QQueue<T>** - Контейнеры FIFO

#### Файловый ввод/вывод
- **QFile** - Операции с файлами
- **QDir** - Операции с каталогами
- **QStandardPaths** - Стандартные расположения путей
- **QFileInfo** - Информация о файлах

### 2. Модуль Widgets (QtWidgets)

#### Фреймворк главного окна
- **QMainWindow** - Главное окно приложения
- **QMenuBar** - Управление меню
- **QStatusBar** - Отображение строки состояния
- **QToolBar** - Управление панелями инструментов

#### Управление компоновкой
- **QVBoxLayout** - Вертикальная компоновка
- **QHBoxLayout** - Горизонтальная компоновка
- **QGridLayout** - Сеточная компоновка
- **QTabWidget** - Интерфейс с вкладками

#### Общие виджеты
- **QLabel** - Отображение текста и изображений
- **QPushButton** - Кликабельные кнопки
- **QLineEdit** - Ввод текста
- **QTextEdit** - Редактирование форматированного текста
- **QTableWidget** - Отображение таблиц
- **QTreeWidget** - Отображение иерархических данных
- **QProgressBar** - Индикация прогресса

#### Классы диалогов
- **QDialog** - Базовый класс диалогов
- **QMessageBox** - Диалоговые окна сообщений
- **QFileDialog** - Диалоги выбора файлов
- **QInputDialog** - Диалоги ввода

### 3. Модуль Network (QtNetwork)

#### Сетевое взаимодействие TCP/IP
- **QTcpSocket** - TCP клиентский сокет
- **QTcpServer** - TCP серверный сокет
- **QHostAddress** - Работа с IP-адресами
- **QNetworkInterface** - Информация о сетевых интерфейсах

#### Конфигурация сети
- **QNetworkConfigurationManager** - Конфигурация сети
